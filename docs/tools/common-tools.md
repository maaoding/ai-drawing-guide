# 常用工具

一句话：工具负责提供界面和流程，模型负责生成能力。选工具要看你想要简单使用还是精细控制。

## Web 平台

Web 平台通常开箱即用，不需要本地显卡。

适合：

- 快速体验。
- 不想安装环境。
- 简单生成海报、头像、插画。

注意：

- 费用、队列和额度限制。
- 平台内容规则。
- 生成图片和上传素材的隐私条款。

## Stable Diffusion WebUI

常见的本地 AI 绘画界面。适合按钮式操作和大量插件生态。

适合：

- 初学者快速上手。
- 使用传统 SD1.5 / SDXL 工作流。
- 大量现成插件和教程。

## ComfyUI

节点式工作流工具。适合精细控制和复杂流程。

适合：

- 学习模型组成和数据流。
- 搭建图生图、局部重绘、放大、参考图控制。
- 复用和分享工作流。

缺点是刚开始会觉得复杂，但理解后更容易定位问题。

## 模型管理工具

模型越来越多后，需要关注：

- 模型文件放在哪里。
- 文件名是否清楚。
- LoRA、VAE、文本编码器是否分目录。
- 是否记录模型来源和许可证。

建议给每个重要模型保留来源链接和说明，避免以后不知道它是什么。

## 图像后期工具

AI 出图不等于最终成品。常见后期包括：

- 裁剪。
- 调色。
- 修手。
- 去瑕疵。
- 添加文字排版。
- 放大和锐化。

常见工具包括 Photoshop、Affinity Photo、Krita、Photopea、GIMP 等。

## 常见误区

- 以为换工具就能解决模型能力问题。
- 只收集模型，不整理来源和版本。
- 忽略后期。公开发布作品通常需要编辑和筛选。
