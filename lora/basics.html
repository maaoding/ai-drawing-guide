LoRA 入门 ​

一句话：LoRA 是挂在底模上的小型附加模型，用来增强某个角色、画风、服装、姿势或概念。

LoRA 能做什么 ​

常见 LoRA 类型：

  • 角色 LoRA：让模型画某个固定人物或原创角色。
  • 画风 LoRA：让画面接近某种绘画、动画或设计风格。
  • 服装 LoRA：增加特定衣服、制服、盔甲或饰品。
  • 姿势 LoRA：强化某类动作或构图。
  • 细节 LoRA：增强质感、线条、手部或画面细节。
  • 概念 LoRA：学习某个物体、场景、材质或设计语言。

LoRA 为什么不能乱用 ​

LoRA 是在某个底模生态上训练出来的。它学到的是“如何修改这个底模内部的某些权重方向”。

因此：

  • SD1.5 LoRA 通常不适合 SDXL。
  • SDXL LoRA 通常不适合 FLUX。
  • Pony LoRA 优先给 Pony 系模型。
  • FLUX.1 LoRA 不要默认给 FLUX.2。
  • Z-Image LoRA 优先给 Z-Image 系模型。

权重怎么调 ​

LoRA 权重表示它对底模影响的强度。

常见起步范围：

  • 角色 LoRA：0.6 到 0.9。
  • 画风 LoRA：0.3 到 0.7。
  • 细节 LoRA：0.2 到 0.5。
  • 多个 LoRA 叠加：每个都适当降低。

如果权重太高，可能出现画面发脏、人体崩坏、风格过强、脸型固定、背景失控等问题。

多个 LoRA 怎么叠加 ​

建议从简单开始：

  1. 先只开一个 LoRA，确认它有效。
  2. 固定 seed。
  3. 再加第二个 LoRA。
  4. 每次只改一个权重。
  5. 记录哪个 LoRA 负责哪个效果。

常见组合：

  • 角色 LoRA + 画风 LoRA。
  • 角色 LoRA + 服装 LoRA。
  • 画风 LoRA + 细节 LoRA。

不要一开始就叠五六个 LoRA。出问题后很难判断是谁造成的。

下载 LoRA 前看什么 ​

重点看：

  • Base model：是否匹配你的底模。
  • Trigger words：需要写哪些触发词。
  • Recommended weight：作者推荐权重。
  • Example prompts：示例提示词。
  • License：能否公开发布或商用。
  • 评论区：是否有版本兼容问题。

常见误区 ​

  • 只看预览图，不看 base model。
  • 把 LoRA 当成完整模型。LoRA 必须配底模。
  • 权重越高越好。过高通常会破坏画面。
  • 多个 LoRA 越多越强。叠加会互相干扰。
