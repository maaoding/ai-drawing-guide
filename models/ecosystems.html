常见模型生态 ​

一句话：选择模型时，不只看画质，还要看生态、显存、LoRA、教程、工作流和许可证。

对比表 ​

生态主要特点适合场景注意事项
SD1.5资源最多、显存低、老教程多老插件、低显存、经典工作流基础分辨率和画质较旧
SDXL画质更好，1024 分辨率常见通用绘画、插画、设计LoRA 需 SDXL 兼容
Pony二次元、角色、标签体系强动漫角色、姿势、风格 LoRA需要理解 score tag 和生态规则
Illustrious二次元插画生态强插画、角色、Danbooru 风格标签不要默认与 Pony 混用
NoobAI二次元标签体系动漫插画、角色 LoRA注意版本和 LoRA 兼容性
FLUX语言理解和质感较强真实感、设计、复杂提示词工作流和 LoRA 生态不同于 SDXL
Qwen-Image中文、多模态、文字渲染方向中文提示词、图像理解、文字相关任务模型较大，配置要求可能更高
Z-Image新一代图像生成生态速度、文字理解、中文或设计类任务生态仍在发展，教程和 LoRA 较少

初学者怎么选 ​

如果你主要学习概念和工具：

  • 低显存或老教程：SD1.5。
  • 通用高质量图片：SDXL。
  • 动漫、角色、LoRA：Pony / Illustrious / NoobAI。
  • 新模型体验和自然语言提示词：FLUX / Qwen-Image / Z-Image。

如果你只是想先学会 ComfyUI 或 WebUI，建议先固定一个生态，不要同时混用太多模型。

看模型页面时重点看什么 ​

下载模型前先看：

  • Base model：它基于哪个生态。
  • Recommended settings：推荐尺寸、步数、CFG、采样器。
  • Trigger words：是否需要特定触发词。
  • License：能否商用，是否有限制。
  • Example prompts：作者示例怎么写。
  • Required files：是否需要额外 VAE、文本编码器或工作流。

常见误区 ​

  • 只看排行榜。热门不代表适合你的用途。
  • 忽略许可证。公开发布作品、商用、训练 LoRA 时尤其要看。
  • 不看版本。模型、LoRA 和工作流经常按版本绑定。
  • 把“模型生态”当成“单个文件”。新模型可能由多个文件组成，例如底模、文本编码器和 VAE。
